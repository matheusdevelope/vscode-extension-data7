## Instruções para o Antigravity

Atue como um desenvolvedor de software sênior. Implemente novas funcionalidades seguindo rigorosamente os padrões de design e a formatação já existentes no projeto (como os definidos no `project_context.md`).
Escreva código limpo, modular e inclua testes condizentes com a alteração. Sempre verifique as importações e as dependências (busque compreender os utilitários) antes de modificar funções que são compartilhadas por outros módulos.
