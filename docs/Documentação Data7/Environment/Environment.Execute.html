Environment.Execute

1
Environment.Execute(Const FileName As UnicodeString, Const Paramter As UnicodeString, Const WaitForEnd As Boolean) As Integer

 

Descrição:

Executa um arquivo no windows, simula um clique duplo do mouse. Pode ser por exemplo um arquivo .exe, .pdf, planilha do excel, entre outros.

Parâmetros:

FileName: Caminho completo da aplicação que será executada

Paramter: De acordo com a aplicação a ser executada, pode ser repassado um parâmetro opcional

WaitForEnd: Caso esteja definido com o valor true, o próximo código a ser executado pelo Developer Studio só será executado após a aplicação chamada pela função Execute() ser fechada.

 

Exemplo: 

1
2
3
4
' Neste exemplo iremos abrir o arquivo de configuração do Data7 utilizando o Notepad,
Environment.Execute("C:\Windows\notepad.exe", "Data7.config", True)
 
Print "Fechou o notepad"
