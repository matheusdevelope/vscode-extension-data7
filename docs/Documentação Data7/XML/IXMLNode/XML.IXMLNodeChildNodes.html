XML.IXMLNode.ChildNodes

1
XML.IXMLNode.ChildNodes() As IXMLNodeList

 

Descrição:

Lista os nós filhos deste nó.

IXMLNode usa ChildNodes para acompanhar os subnós de um nó. ChildNodes é uma interface IXMLNodeList, que pode ser usada para acessar nós filhos individuais ou adicionar nós filhos ou excluí-los desse nó. Cada nó filho especifica esse nó como seu nó pai.

Nota: Os atributos não são considerados nós filhos de um nó. Para obter os valores dos atributos desse nó, use a propriedade Attributes.

Dim frm As Forms.Form = New Forms.Form()
Dim vXML As XML.TXMLDocument = New XML.TXMLDocument(frm)
Dim tagNFe As XML.IXMLNode
 
vXML.LoadFromFile("D:\NFe Compra.xml")
 
tagNFe = vXML.DocumentElement.SelectSingleNode("NFe")
 
Print tagNFe.ChildNodes().Nodes(0).GetNodeName()
Print tagNFe.ChildNodes().Nodes(1).GetNodeName()
 
vXML.Free()
frm.Free()
