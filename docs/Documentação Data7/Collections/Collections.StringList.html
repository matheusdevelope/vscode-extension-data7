Namespace Collections.StringList

 

 Function  Descrição
 Add  Adiciona uma string à lista de strings e retorna o índice do item na lista.
 AddObject  Adiciona uma sequência à lista e associa um objeto à sequência.
 Equals  Compara duas strings.
 IndexOf  Utilize IndexOf para obter a posição da primeira ocorrência da sequência S ou de uma sequência que difere de S apenas por caso.
 IndexOfObject  Retorna o índice da primeira string na lista associada a um determinado objeto.
 Objects  Use a propriedade Objects para obter ou definir o objeto associado à string na posição indicada pelo Index .

 

 Sub  Descrição
 AddStrings  Adiciona strings de outro objeto TStrings à lista.
 Append  Adiciona uma string à lista de strings. 
 Assign  Use Assign para definir o valor do objeto TStrings de outro objeto.
 BeginUpdate  Chame BeginUpdate antes de modificar diretamente as sequências de caracteres na lista e EndUpdate depois.
 Clear  Limpa a lista de Strings.
 Delete  Descendentes de TStrings implementam um método Delete para remover uma seqüência especificada da lista.
 EndUpdate  EndUpdate é chamado automaticamente por qualquer propriedade ou método que altera a lista de strings.
 Exchange  Utilize o Exchange para reorganizar as seqüências de caracteres na lista.
 Insert  Insere uma string na lista na posição especificada pelo Índice .
 InsertObject  Insere uma string na lista na posição especificada e a associa a um objeto.
 LoadFromFile  Preenche uma lista de strings com o dados de um arquivo físico.
 Move  Altera a posição de uma sequência na lista.
 SaveToFile  Salva as strings em um arquivo de texto.

 

 Property  Descrição
 Capacity  Utilize para definir ou obter a capacidade máxima de Strings que poderão ser adicionadas à lista.
 CommaText  Utilize para obter ou definir a lista por meio de uma string.
 IndexOfName  Utilize IndexOfName para localizar a primeira ocorrência de um par nome-valor em que a parte do nome é igual ao parâmetro Name ou difere     apenas no caso.
 NameValueSeparator  As seqüências que contêm o caractere NameValueSeparator são consideradas pares nome-valor.
 Names  Quando a lista de strings do objeto TStrings inclui cadeias que são pares nome-valor, leia Names para acessar a parte do nome de   uma cadeia.
 OwnsObjects  A propriedade OwnsObjects especifica se a lista de cadeias possui ou não os objetos armazenados.
 Sorted  Ordena a lista de strings por ordem ascendente.
 Strings  Referencia as seqüências de caracteres na lista por suas posições.
 Values  Quando a lista de strings do objeto TStrings inclui strings que são pares nome-valor, use Values para obter ou definir a parte do valor de uma cadeia associada a uma parte de nome específica.
