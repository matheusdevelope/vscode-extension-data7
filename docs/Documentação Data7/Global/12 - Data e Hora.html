12 - Data e Hora

Nº do Artigo: 209 | Avaliação: 5/5 de 1 votos | Última atualização: Tue, May 28, 2019 at 6:00 PM

A maioria dos softwares que você escreve precisa implementar alguma forma de função de data, retornando a data e hora atuais. As datas fazem parte da vida cotidiana e torna-se fácil trabalhar com elas sem pensar. O Developer Studio também fornece ferramentas poderosas para aritmética de datas que facilitam a manipulação de datas.

O tipo de dados TDateTime contém valores de data, valores de hora ou valores de data e hora.

' Retorna true se a data for igual a data informada no parâmetro

Dim datahora As TDateTime
Dim horaInicial As TDateTime
Dim horaFinal As TDateTime
Dim dataInicial As TDateTime
Dim dataFinal As TDateTime
 
'-------------------------------------------------------------------------------
'Declaração das variáveis-------------------------------------------------------
'-------------------------------------------------------------------------------
' "Now()" define a data atual para a variável data
datahora = DateTime()
 
'StrToDateTime Converte uma String para o tipo DateTime
horaInicial = StrToDateTime("30/04/2019 14:00:10")
 
'StrToDateTime Converte uma String para o tipo DateTime
horaFinal = StrToDateTime("30/04/2019 16:30:58")
 
'StrToDateTime Converte uma String para o tipo Date
dataInicial = StrToDate("10/01/2018")
 
'StrToDateTime Converte uma String para o tipo Date
dataFinal = StrToDate("20/03/2019")
 
Print "Função ToString() e seus possíveis parâmetros:"
' y = Últimos 2 dígitos do ano
Print "y: " & datahora.ToString("y")
Print "yy: " & datahora.ToString("yy")
 
' yyyy = Ano com 4 dígitos
Print "yyyy: " & datahora.ToString("yyyy")
 
' m = Mês sem o zero à esquerda
Print "m: " & datahora.ToString("m")
 
'mm = Mês com dois digítos (zero à esquerda)
Print "mm: " & datahora.ToString("mm")
 
' mmm = Mês utilizando nome curto Ex: (Jan)
Print "mmm: " & datahora.ToString("mmm")
 
' mmmm = Mês utilizando nome longo Ex: (January)
Print "mmmm: " & datahora.ToString("mmmm")
 
' d = Dia sem o zero à esquerda
Print "d: " & datahora.ToString("d")
 
' dd = Dia com dois digítos (zero à esquerda)
Print "dd: " & datahora.ToString("dd")
 
' ddd = Dia utilizando nome curto Ex: (Seg)
Print "ddd: " & datahora.ToString("ddd")
 
' dddd = Dia utilizando nome longo (Segunda)
Print "dddd: " & datahora.ToString("dddd")
 
' ddddd     = Dia no formato curto
Print "ddddd: " & datahora.ToString("ddddd")
 
' dddddd = Dia no formato longo
Print "dddddd: " & datahora.ToString("dddddd")
 
' c = Dia e hora no formato curto
Print "c: " & datahora.ToString("c")
 
' h = Hora com dois dígitos (sem zero à esquerda)
Print "h: " & datahora.ToString("h")
 
' hh = Hora com dois dígitos (com zero à esquerda)
Print "hh: " & datahora.ToString("hh")
 
' n = Minuto com dois dígitos (sem zero à esquerda)
Print "n: " & datahora.ToString("n")
 
' nn = Minuto com dois dígitos (com zero à esquerda)
Print "nn: " & datahora.ToString("nn")
 
' s = Segundo com dois dígitos (sem zero à esquerda)
Print "s: " & datahora.ToString("s")
 
' ss = Segundo com dois dígitos (com zero à esquerda)
Print "ss: " & datahora.ToString("ss")
 
' z = Milisegundo sem zero à esquerda
Print "z: " & datahora.ToString("z")
 
' zzz = Milisegundo com 3 dígitos
Print "zzz: " & datahora.ToString("zzz")
 
' t = Hora no formato curto
Print "t: " & datahora.ToString("t")
 
' tt = Hora no formato longo
Print "tt: " & datahora.ToString("tt")
 
' am/pm = Após a hora, exibe as letras am ou pm
Print "am/pm: " & datahora.ToString("hham/pm")
 
' a/p = Após a hora, exibe a letra a ou p
Print "a/p: " & datahora.ToString("hha/p")
 
' ampm = Exibe a hora no formato am/pm
Print "ampm: " & datahora.ToString("hhampm")
 
' h:n:s.z = Exibe a hora sem zeros
Print "h:n:s.z: " & datahora.ToString("h:n:s.z")
 
' hh:nn:ss.zzz = Exibe a hora com zeros
Print "hh:nn:ss.zzz: " & datahora.ToString("hh:nn:ss.zzz")
