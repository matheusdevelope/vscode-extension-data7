3 - Tipos de Dados

Nº do Artigo: 200 | Avaliação: Não avaliado | Última atualização: Thu, May 9, 2019 at 5:41 PM

Tipos de dados referem-se a um sistema extensivo usado para declarar variáveis ​​ou funções de diferentes tipos. O tipo de variável determina quanto espaço ocupa no armazenamento e como o padrão de bits armazenado é interpretado.

Tipos de dados disponíveis no Developer Studio

O Developer Studio fornece uma ampla gama de tipos de dados. A tabela a seguir mostra todos os tipos de dados disponíveis:

 Tipo de   dados Alocação de Armazenamento Faixa de valor
 Boolean    Verdadeiro ou falso
 Byte  1 byte  0 a 255 (sem assinatura)
 Char  2 bytes  0 a 65535 (sem assinatura)
 Date  8 bytes  0:00:00 (meia-noite) de 1º de janeiro de 0001 a 23:59:59 de 31 de dezembro de 9999
 Decimal  16 bytes  0 a +/- 79.228.162.514.264.337.593.543.950.335 (+/- 7,9 ... E + 28) sem ponto decimal; 0 a +/- 7,9228162514264337593543950335   com 28 lugares à direita da casa decimal
 Double  8 bytes

 -1.79769313486231570E + 308 a -4.94065645841246544E-324, para valores negativos

 4.94065645841246544E-324 a 1.79769313486231570E + 308, para valores positivos
 Integer  4 bytes  -2.147.483.648 através de 2.147.483.647 (assinado)
 Long  8 bytes  -9,223,372,036,854,775,808 a 9,223,372,036,854,775,807 (assinada)
 TObject

 4 bytes na plataforma de 32 bits

 8 bytes na plataforma de 64 bits

  Qualquer tipo pode ser armazenado em uma variável do tipo Object
 Short  2 bytes  -32.768 a 32.767 (assinado)
 Single  4 bytes

 -3,4028235E + 38 a -1,401298E-45 para valores negativos;

 1,401298E-45 a 3,4028235E + 38 para valores positivos
 String  Depende da plataforma de   implementação  0 a aproximadamente 2 bilhões de caracteres Unicode

As funções de conversão de tipo no Developer Studio

O Developer Studio fornece as seguintes funções de conversão de tipo em linha:

 Funções e Descrição

 CBool ​​(expressão)

 Converte a expressão em tipo de dados booleano.

 CByte (expressão)

 Converte a expressão em tipo de dados Byte.

 CChar (expressão)

 Converte a expressão em tipo de dados Char.

 CDate (expressão)

 Converte a expressão para o tipo de dados Data

 CDbl (expressão)

 Converte a expressão em Double type de dados.

 CDec (expressão)

 Converte a expressão em tipo de dados Decimal.

 CInt (expressão)

 Converte a expressão em tipo de dados Integer.

 CLng (expressão)

 Converte a expressão em tipo de dados Long.

 CSByte (expressão)

 Converte a expressão em tipo de dados SByte.

 CSng (expressão)

 Converte a expressão em tipo de dados Único.
