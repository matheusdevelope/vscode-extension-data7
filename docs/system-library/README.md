# System Library do Data7 — Documentação

> Conjunto de definições nativas reconhecidas pela extensão **Data7 Dev Studio**.
> Use este conteúdo como contexto para agentes de IA no VS Code ou como referência rápida.

## Namespaces incluídos

- [`Collections`](./Collections.md) — Namespace nativo do Data7 para estruturas de dados e coleções (StringList, etc.).
- [`Data7`](./Data7.md) — Namespace global contendo utilitários fornecidos pelo ERP Data7.
- [`Drawing`](./Drawing.md) — Namespace nativo do Data7 para recursos gráficos e de desenho (TCanvas, TPen, etc.).
- [`Environment`](./Environment.md) — Utilitários sobre o ambiente de execução e Windows.
- [`Forms`](./Forms.md) — Namespace para manipulação da interface de formulários e caixas de diálogo do ERP.
- [`IO`](./IO.md) — Namespace para manipulação do sistema de arquivos e diretórios.
- [`Net`](./Net.md) — Namespace do ERP Data7 para acesso a serviços de rede (FTP, sockets, etc.). Atualmente expõe a classe `TFTP` para conexão e transferência de arquivos via FTP.
- [`SQL`](./SQL.md) — Namespace contendo utilitários de acesso e execução de comandos SQL no banco de dados do ERP.
- [`System`](./System.md) — Namespace básico do RTL Delphi — funções de memória, matemática, strings, conversão de tipos e constantes Variant.
- [`System.Classes`](./System.Classes.md) — Namespace contendo as classes bases de persistência e strings do Delphi (TObject, TPersistent, TStrings, TStringList).
- [`XML`](./XML.md) — Namespace do ERP Data7 para manipulação e geração de documentos XML.
